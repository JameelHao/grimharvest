# ⚔️ Grimharvest

> *Reap the horde. Survive the night.*

A medieval horde-survival game (割草游戏) in the spirit of Vampire Survivors — you are the realm's last reaper, mowing down endless waves of the risen dead with auto-attacking weapons, evolving builds, and pure footwork.

## 🎮 Core Loop

- **15-minute survival runs** — enemy density scales relentlessly, a Boss rises in the final minute
- **Auto-attack, manual movement** — weapons fire on their own; the game is positioning + build choices
- **Kill → XP gems → level up → pick 1 of 3** — every level-up is a draft from weapons and passives

## 🗡️ Build System

- 4 weapon slots + 4 passive slots — forced trade-offs
- **Weapon evolutions**: max a weapon + hold its paired passive → open a chest to evolve it into a transformed form
- Synergy tags (projectile / area / sustained) so both specialist and mixed builds work

## 💀 Enemies

Differentiated by *behavior*, not just stats: chasers, chargers, ranged spitters, encircling rings, and chest-dropping elites.

## 🛠️ Tech

TypeScript + Canvas, deployed on GitHub Pages. Object pooling + spatial hashing to keep thousands of enemies at 60fps.
