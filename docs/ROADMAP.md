# Grimharvest 开发路线

机制详见 [DESIGN.md](DESIGN.md)。每个里程碑结束时 `main` 必须可开局、不报错（CLAUDE.md 约定）。

| 里程碑 | 内容 | 状态 |
|--------|------|------|
| **M0 脚手架 + 美术** | Vite+TS、离屏精灵预渲染、PNG 校对工具、主角 reaper + 小怪 husk | ✅ 完成 |
| **M1 引擎地基** | 固定步长循环 + 渲染插值、输入、相机跟随、玩家移动、Y 排序绘制、敌人追击 | ✅ 完成 |
| **M2 战斗地基** | 可种子化 PRNG、空间哈希、对象池、挥镰自动攻击、敌人受击/死亡**留尸**、玩家血量/受击/死亡重开 | ✅ 完成 |
| **M3 魂收割** | 魂掉落/自动吸附/连击/超时复生为亡者、经验与等级（支柱一） | ✅ 完成 |
| **M4 成长** | 升级三选一（DOM 面板）、月镰/飞镰/使魔(蝙蝠)三武器、被动强化 | ✅ 完成 |
| **M5 经济与潮汐** | Grain/Chaff 双资源、诅咒强化（怨魂兑换+反噬）、恐惧潮汐刷怪、波次配置表（支柱二 & 三） | ✅ 完成 |
| **M6 内容** | 冲撞者/吐息者/环绕者/精英 4 种行为敌人 + 专属像素美术、敌方投射物、Boss 镰影(召唤亡者/HP 条/胜利结算) | ✅ 完成 |
| **M7 打磨与上线** | 每日种子挑战(?seed 可覆盖)、性能压测(按 P 刷 2000 敌)+ FPS 显示、GitHub Actions 部署 workflow | ✅ 完成（待推送上线） |

> 进化/留种(成熟与留种)尚未实现，留作 M8 候选。
