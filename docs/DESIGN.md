# Grimharvest 机制设计

> 一句话定位：这不是「站撸吸经验」，而是**割 → 脱粒 → 抢收**——你亲手把尸体收割成净魂，慢一步它们就复生反扑。「收割」是玩法，不只是题材。

设计原则照旧遵守 [CLAUDE.md](../CLAUDE.md)：所有数值进 `src/data/`，逻辑不碰数值。

---

## 三大支柱

### 支柱一 · 魂收割 The Soul Harvest（招牌机制）

把 Vampire Survivors 的「被动吸经验」改成**主动抢收灵魂、慢一步就反扑**：

- 敌人死亡掉落一颗**魂 Soul**（不是直接加经验）。
- 魂进入玩家**吸附半径**内会自动飞向主角 = **经验**；连续吸魂叠加**收割连击 Harvest Combo**，连击越高单颗经验越多。
- 魂有 **~5 秒收割窗口**：没被吸到就**就地复生为亡者 Risen**——更快、更硬、更具攻击性（且亡者死后又掉魂，形成连锁压力）。

**为什么有趣**：经验是会跑路、还会反咬的资源。逼你主动往魂堆/尸堆里钻去抢收；贪多清不过来，满地的魂就一起站起来反扑。吸附半径做成可升级被动，给 build 一个成长维度。100% 贴合「收割」主题。

### 支柱二 · 净魂与怨魂（暗黑契约式 build 经济）

- 两种灵物：**净魂 Grain**（脱粒所得，喂等级）与**怨魂 Chaff**（复生亡者被二次斩杀掉落的「受污」灵物）。
- 升级三选一照常消耗 Grain；但你可以**消耗 Chaff** 解锁一排**诅咒强化 Cursed picks**：强力但带反噬（如 +50% 伤害但净魂半衰更快 / 召唤随从但持续掉血）。
- 于是「放任尸体复生」成了一种**资源策略**：稳健流脱粒求 Grain，激进流养尸刷 Chaff 走诅咒 build。

### 支柱三 · 恐惧潮汐 Dread Tide（替代纯时间曲线的节奏）

- **恐惧值**由「同屏敌人数 + 你的击杀速率」共同驱动，而非只看时间。
- 恐惧越高：刷怪越密、精英概率越高、掉落越肥；越稳：场面变缓。
- 你可以**主动养潮**（憋一波大丰收）或**割草降潮**（保命）。Boss 仍在 15:00 降临，但潮汐决定这一路多凶、多肥。

## 武器进化重构 · 成熟与留种 Mature / Go-to-Seed

- 选到的武器是一颗**种子**：初始弱，靠喂净魂 / 击杀逐渐**成熟**。
- 满级开宝箱二选一：
  - **收获 Harvest** = 进化成形态（沿用 README 的进化设定）；
  - **留种 Go-to-Seed** = 献祭该武器，换一次性强力效果 + 释放一个槽位。
- 进化条件、成熟曲线全部写在 data 表，逻辑不碰数值。

## 敌人 · 用行为区分（呼应 README）

| 敌人 | 行为钩子 |
|------|----------|
| 腐生子 Husk | 基础追击杂兵（已建模） |
| 冲撞者 Charger | 蓄力后直线猛冲，逼你侧身闪 |
| 吐息者 Spitter | 远程抛射，留下毒地 |
| 环绕者 Ringer | 成群结环包夹，压缩你的脱粒空间 |
| 收尸人 Elite | 必掉宝箱，但身边尸壳**复生更快**，是清场压力源 |

共性钩子：所有近战单位死亡都留尸 → 持续喂养脱粒循环。

## Boss · 镰影 The Harvestman

15:00 降临的不是更大的怪，而是**来收割你的死神**：它会**和你抢脱粒**——主动复活并吞噬场上尸壳壮大自己。你得和它抢这一季的收成。镜像玩家机制，收尾点题。

## 每日种子挑战

全程走自实现的可种子化 PRNG（CLAUDE.md 既定）：刷怪、三选一、掉落都吃同一种子；每日固定种子打榜。

## 飞轮：机制如何互相咬合

```
脱粒(支柱一) 产出 Grain/Chaff
      │
      ▼
build 经济(支柱二) 决定你的杀伤
      │
      ▼
杀戮速率回灌 恐惧潮汐(支柱三)
      │
      ▼
潮汐决定尸体产量 ── 回到脱粒
```

一个正反馈飞轮，且每一环都给玩家一个「贪还是稳」的选择。

## 数值落点（`src/data/`）

| 文件 | 内容 |
|------|------|
| `enemies.ts` | 血量 / 速度 / 行为 / 尸壳复生时限 / 掉落表 |
| `waves.ts` | 时间 × 恐惧 的刷怪节奏表 |
| `weapons.ts` | 基础数值 / 成熟曲线 / 进化 & 留种条件 |
| `upgrades.ts` | 三选一权重 + 诅咒强化表（含反噬） |
| `balance.ts` | 连击倍率曲线、恐惧潮汐系数、灵物半衰 |
| `sprite-data.js` | 精灵美术（已建：reaper / husk） |
